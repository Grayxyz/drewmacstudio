Procedural Neon System

Procedrual Lighting System in Houdini

I started investigating Houdini in 2018 after seeing a Blender Con talk by Andrew Price in which he showed some of the capabilities of procedrual surface operators in Houdini. One thing that has always frustrated me about traditional modeling in Maya or Cinema 4D was the destructive nature of it. While Maya preserves actions in the history of the model, there was never a great solution to select geomoetry dynamically

In this project I designed the system such that any vector file could be imported, and the system would create a neon light from it and render it in Redshift. The system also employs point attributes to animate the color of the light for situations in which an animated LED-type system is desireable.
